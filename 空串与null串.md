# 空串与null串
Java中的字符串`String`中的空串与`null`串有着很大的不同,空串""是长度为的字符串,可以通过下面的方式来判断一个字符串是否为空。
```java
if(str.length() == )
```
或者
```java
if(str.equals(""))
```
但是`null`串完全不一样,表示目前没有任何对象与该变量关联,如果在一个`null`值上调用方法就会产生空指针异常,如果想要检查一个字符串是否为空或者是否为`null`,可以采用下面的方式:
```java
if(str != null && str.length() != 0)
```
